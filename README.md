SOAL MUDAH 1
1. Program menghitung biaya pengiriman barang
2. Input: berat, jarak, dan ukuran (p × l × t)
3. Jika jarak < 10 km - biaya = 4250 × berat
4. Jika jarak > 10 km - biaya = 6000 × berat
5. Jika volume > 100 - tambah biaya Rp 50.000
6. Output: Total biaya pengiriman

SOAL MUDAH 2
1. Program ini menentukan apakah suatu bilangan genap atau ganjil
2. Input: pengguna memasukkan bilangan (bilangan1)
3. Jika bilangan1 % 2 == 0, maka bilangan genap (bisa dibagi 2 tidak ada sisa)
4. Jika tidak, maka bilangan ganjil (hasil dibagi 2 menyisakan 1)
5. Output: menampilkan keterangan apakah bilangan tersebut genap atau ganjil dengan alasannya

SOAL SEDANG 1
1. Program ini untuk hitung faktorial dari angka yang kita masukin
2. Waktu dijalankan, program bakal minta input angka (n)
3. Nilai awal faktorial dikasih 1 dulu
4. Terus pakai perulangan for, dari 1 sampai n, tiap langkah faktorial dikali sama i
5. Habis loop selesai, hasil akhirnya ditampilin ke layar

SOAL SEDANG 2
1. Program ini untuk hitung volume tabung 
2. Pertama, program minta kamu masukin jari-jari (r) dan tinggi (t) tabung
3. Nilai phi dijadikan 3.1415
- Rumus volume tabungnya:
- volume = phi * r * r * t
4. Terus hasilnya ditampilin di layar

SOAL SULIT 1
1. Program ini untuk hitung rata-rata nilai siswa
2. Pertama, masukin jumlah siswa
3. Lalu program minta nilai tiap siswa satu per satu
4. Semua nilai dijumlahkan jadi total Nilai
5. Setelah itu hitung rata-ratanya pakai rumus:
- rataRata = total Nilai / jumlah Siswa
6. Terakhir, hasil rata-rata ditampilin dengan dua angka di belakang koma

SOAL SULIT 2

1. Program ini untuk menjumlahkan dua matriks (A dan B)
2. Pertama, masukin jumlah baris dan jumlah kolom matriks
3. Program bikin tiga matriks yaitu:
- matriksA, matriksB, dan hasil
4. Terus kamu masukin elemen-elemen matriks A dan B satu-satu
5. Setelah itu, program menjumlahkan tiap elemen yang posisinya sama
6Hasil akhirnya disimpan di matriks hasil
